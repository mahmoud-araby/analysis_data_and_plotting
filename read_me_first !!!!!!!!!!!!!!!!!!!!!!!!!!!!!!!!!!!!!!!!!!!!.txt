this program deals with .txt files only 
this program uses third party program for plotting called gnuplot   ----
to use the plotting function you must do next :::::

1- download gnuplot program from https://sourceforge.net/projects/gnuplot/
2- setup gnuplot program -
3- in steup process you must select ((( Add application directory to your PATH environment variable ))) or the program won`t work .
4- finish the installation process 
5- restart your programming environment .
6- your plotting functions will work just fine .



note :::: the program will work correctly if you didn`t use gnuplot but it won`t show plotting data to the screen ;
